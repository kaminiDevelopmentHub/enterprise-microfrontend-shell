# 🚀 Enterprise UI Shell (Micro-Frontend Architecture)

This project demonstrates a **scalable frontend architecture** using **Micro-Frontends (MFA)** with Webpack Module Federation.

The application is built as a **Shell (Host App)** that dynamically loads independent micro-applications like Dashboard and Orders at runtime.

---

## 🧠 Architecture Overview

- Shell App (Host)
- Remote Apps:
  - Dashboard App
  - Orders App

Each micro-application is:
- Independently developed
- Independently deployed
- Dynamically integrated

---

## ⚙️ Tech Stack

### Frontend
- React.js (TypeScript)
- Redux Toolkit
- React Router
- Webpack Module Federation
- CRACO (for custom webpack config)

### Architecture
- Micro-Frontend (MFA)
- Modular & Scalable UI Design
- API Abstraction Layer

---

## 🚀 Features

- Dynamic loading of remote applications
- Scalable micro-frontend architecture
- Global state management using Redux
- Lazy loading for performance optimization
- Reusable component-based design
- Independent deployment support
